# laravel-api-boilerplate
Laravel 7 API boilerplate

- git clone https://github.com/webturtles18/laravel-api-boilerplate.git

## Installation

```sh
composer install
```

- Create **.env** file by copy **.env.example** file.

```sh
php artisan migrate
```
